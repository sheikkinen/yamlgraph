## Completed Items

### Helper Script: `run_generator.py`

```bash
# Generate a new graph
python run_generator.py "Create a Q&A pipeline"

# Generate to specific directory
python run_generator.py -o ./my-graph "Create a router pipeline"

# Generate, lint, and run
python run_generator.py --run "Create a simple pipeline"

# Run existing graph with input
python run_generator.py --run-only -o ./my-graph --input question="What is AI?"

# Lint only
python run_generator.py --lint-only -o ./my-graph

# Clean and regenerate
python run_generator.py --clean -o ./my-graph "Create a new pipeline"
```
